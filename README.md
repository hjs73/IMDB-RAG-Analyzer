# IMDB-RAG-Analyzer
Implements an RAG pipeline using IMDB Movie Review dataset to build an interactive IMDB Movie Review Analyzer Streamlit app

Step 1: Install all packages used in git_imdb_rag.py and git_streamlit_imdb.py
(pandas, numpy, openai, faiss-cpu, time, streamlit)

Step 2: Download the IMDB dataset here: https://ai.stanford.edu/~amaas/data/sentiment/

Read in the dataset from the file path you have it stored

Step 3: For the API key, replace "insert_api_key_here" with your own OpenAI API key

Step 4: Run git_imdb_rag.py

Note, this may take several minutes

Step 5: Run git_streamlit_imdb.py

Step 6: To launch app:
streamlit run git_streamlit_imdb.py
